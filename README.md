# 放課後等デイサービス ひかり｜LP制作

このリポジトリは、放課後等デイサービス「ひかり」の紹介用ランディングページ（LP）です。  
温かみのある色合い・柔らかいUI・児童支援の現場に寄り添った構成を意識して制作しました。

---

## 🌈 コンセプト

- **保護者が安心できるデザイン**
- **子どもに寄り添う温かい色味**
- **読みやすく、やさしいレイアウト**
- **放デイの支援内容が直感的に伝わる構成**

現場経験を活かし、  
「初めて見る保護者が不安なく読み進められるLP」を目指しています。

---

## 🛠 使用技術

- **HTML5**
- **CSS3**
  - パステルカラーのテーマ
  - 波形背景（SVG）
  - ぷにっとしたボタンデザイン
  - セクションごとの背景色切り替え
- **JavaScript**
  - IntersectionObserver を用いたフェードインアニメーション

---

## 📄 主なセクション構成

- ファーストビュー（温かいグラデーション）
- 対象となるお子さま
- 支援内容（アイコン付きカード）
- 1日の流れ（タイムライン）
- 事業所の特徴
- スタッフ紹介
- よくある質問（FAQ）
- 見学予約（CTA）

---

## 🎨 デザインのポイント

- **温かみのあるクリーム × ピーチ × オレンジの配色**
- **丸みのあるカード・ボタンで安心感を演出**
- **波形背景で柔らかい区切り**
- **スマホでも読みやすいレスポンシブ構成**
